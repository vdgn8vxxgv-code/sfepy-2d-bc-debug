# SFEpy 2D — интерактивная отладка граничных условий (BC)

Интерактивный учебно-практический проект на Python + SfePy для пошаговой
отладки и наглядного понимания граничных условий в задаче стационарной
теплопроводности в 2D (геотехническая/насыпная геометрия).

Фокус: как именно **BC меняют решение**, как их правильно задать на уровне
матриц \(K\) и правой части, и как это всё визуализировать.

Версия: **1.0 (этапы BC#1–4 полностью реализованы и протестированы)**  
Дата: **Январь 2026**


## 1. Что это такое, в одном абзаце

Проект демонстрирует, как по шагам добавлять граничные условия в модель
теплопроводности 2D в SfePy для трапециевидной насыпи на грунте:

- Сначала только **Дирихле** на дне (BC#1).
- Затем добавляются **конвективные** условия (Робин) на откосах (BC#2–3).
- Затем усиливается охлаждение **на вершине насыпи** (BC#4).

Каждый шаг — отдельный скрипт: легче отлаживать, контролировать и
понимать физику. Визуализация (Matplotlib) показывает геометрию, BC и
температурные поля с профилями и статистикой.

9. Если что-то идёт не так
См. разделы «Ошибки / отладка» в:

STEP_BY_STEP_DEBUG_GUIDE.md,

bc_reference_complete.txt.

Типичные проблемы:

не найден модуль (ModuleNotFoundError) → установить зависимости;

сингулярная матрица → некорректно заданы BC (двойное условие
на один узел, забыто обнуление строки при Дирихле и т.п.);

странные поля температур → проверить параметры, единицы измерения,
знаки потоков и т.д.

При необходимости можно сделать минимальный пример и адаптировать скрипты
под другую геометрию/сети.
